# 📧 Spam Mail Detection using SVM

## 📌 Overview

This project focuses on detecting spam emails using Machine Learning. It classifies emails as **Spam** or **Not Spam (Ham)** using the **Support Vector Machine (SVM)** algorithm.

The system analyzes email text and predicts whether it is spam, helping reduce unwanted messages and improving email security.

---

## 🎯 Objective

* Build a spam detection model using SVM
* Classify emails automatically
* Achieve high accuracy in prediction

---

## 🧠 Algorithm Used

* Support Vector Machine (SVM)
* Finds optimal hyperplane to separate spam and non-spam

---

## 📂 Dataset

* Name: SMS Spam Collection Dataset
* Source: Kaggle
* Type: Text Data
* Classes:

  * 0 → Not Spam (Ham)
  * 1 → Spam

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 🚀 Project Workflow

1. Load dataset
2. Preprocess text data
3. Convert text to numerical format (TF-IDF)
4. Split dataset into training and testing
5. Train SVM model
6. Predict output
7. Evaluate performance
